# Healthcare SQL Analysis

## 📌 Project Overview

This project demonstrates SQL-based healthcare data analysis using MySQL. It includes database creation, data analysis, and business-oriented SQL queries to generate meaningful insights from hospital, patient, admission, and treatment data.

---

## 🎯 Objectives

- Analyze patient demographics
- Calculate hospital utilization and revenue
- Analyze treatment costs
- Measure patient length of stay
- Apply advanced SQL concepts for healthcare analytics

---

## 🗂 Database Tables

- Patients
- Hospitals
- Admissions
- Treatments

---

## 🛠 SQL Concepts Used

- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (COUNT, SUM, AVG)
- INNER JOIN
- LEFT JOIN
- Subqueries
- Views
- Window Functions (RANK, DENSE_RANK)

---

## 📊 Analysis Performed

- Patient Demographics Analysis
- Hospital Utilization Analysis
- Treatment Cost Analysis
- Length of Stay Analysis
- Advanced Filtering
- Data Combination using Joins
- Subquery Analysis
- Hospital Performance View
- Ranking Hospitals by Revenue
- Ranking Treatments by Frequency
